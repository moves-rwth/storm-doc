Specifies a transition of the memory structure.

The interpretation is that we switch from startState to goalState upon entering one of the specified model states (via one of the specified choices).

Note
If it is not possible to move from startState to goalState, such a transition does not have to be set explicitly.
Parameters
startStatethe memorystate in which the transition starts
goalStatethe memorystate in which the transition ends
modelStatesthe model states that trigger this transition.
modelChoicesif given, filers the choices of the model that trigger this transition.
