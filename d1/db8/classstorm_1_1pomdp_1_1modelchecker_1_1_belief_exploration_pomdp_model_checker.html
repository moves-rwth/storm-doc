class storm::pomdp::modelchecker::BeliefExplorationPomdpModelChecker< PomdpModelType, BeliefValueType, BeliefMDPType >

Model checker for checking reachability queries on POMDPs using approximations based on exploration of the belief MDP.

Template Parameters
PomdpModelTypemodel type of the POMDP
BeliefValueTypetype used for the state probabilities in beliefs
BeliefMDPTypenumber type used for the MDP structure of the belief MDP. BeliefMDPType can differ from BeliefValueType as we might want to have exact values for probabilities in beliefs, but are okay with possible imprecision in the MDP itself.

Allows to generate an under-approximation using a controllable unfolding.

This provides a method for outside tools to control the unfolding for an under-approximation themselves. The unfolding runs until a pausing command is issued. If the unfolding is paused, cut-offs and optionally clipping are applied to obtain an abstraction MDP. This MDP is then checked and the result is saved. The unfolding can then be continued from the state before cut-offs were applied.

Parameters
targetObservationsthe target observations of the objective
mintrue if the objective is to minimise the value
rewardModelNamename of the reward model to be used if one is specified
valueBoundsvalues used for cut-offs and clipping
resultthe struct to store results
