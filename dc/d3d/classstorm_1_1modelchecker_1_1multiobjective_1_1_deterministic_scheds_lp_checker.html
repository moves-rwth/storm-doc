Optimizes in the currently given direction.

If there is a point in the given area, an achievable point p and a value v are returned such that p*w≈q*w≈v, where w is the weight vector and q is an (unknown) optimal point in the overapproximation.

  • if eps is present, the requirement is p*w ≤ q*w ≤ v and |p*w-v| ≤ |eps*w|, and
  • if eps is not present (i.e. empty), the default precision of the underlying LP solver is used. If there is no achievable point in the given area, the returned object remains uninitialized.
Parameters
epsif not empty, defines the precision requirement
