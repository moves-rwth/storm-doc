Construct a new iterator.

We either iterate over all failable BEs or over all dependencies (if dependency is true). For dependencies, we start with non-conflicting dependencies before iterating over conflicting ones. For dependencies, we start with non-conflicting dependencies if they are present.

Parameters
dependencyWhether dependencies should be iterated (or BEs if false).
conflictingWhether conflicting dependencies should be iterated (or non-conflicting if false).
iterBEIterator for BEs.
iterDependencyIterator for Dependencies.
nonConflictEndIterator to end of non-conflicting dependencies.
conflictBeginIterator to begin of conflicting dependencies.
