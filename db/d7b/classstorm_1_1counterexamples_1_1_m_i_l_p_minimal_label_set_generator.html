Computes a (minimal) counterexample with respect to the number of prism commands for the given model and (safety) formula.

If the model satisfies the property, an exception is thrown.

Parameters
mdpAn MDP that is the model in which to generate the counterexample.
formulaPtrA pointer to a safety formula. The outermost operator must be a probabilistic bound operator with a strict upper bound. The nested formula can be either an unbounded until formula or an eventually formula.
