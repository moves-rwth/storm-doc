Check if two given dependencies are conflicting in their resolution, i.e.

check if non-determinism may occur. Note that this is a very conservative check using SMT formulae. We only check if sequences exist, where one of the dependencies is triggered before the other is completely resolved

Parameters
dep1IndexIndex of the first dependency
dep2IndexIndex of the second dependency
timeouttimeout for the solver
Returns
"Sat" if the dependencies are conflicting, "Unsat" if they are not, otherwise "Unknown"
