Computes the minimal command set that is needed in the given model to exceed the given probability threshold for satisfying phi until psi.

Parameters
symbolicModelThe symbolic model description that was used to build the model.  
modelThe sparse model in which to find the minimal command set.
phiStatesA bit vector characterizing all phi states in the model.
psiStatesA bit vector characterizing all psi states in the model.
propertyThresholdThe threshold that is to be achieved or exceeded.
rewardNameThe name of the reward structure to use, or boost::none if probabilities are considerd.
strictBoundIndicates whether the threshold needs to be achieved (true) or exceeded (false).
optionsA set of options for customization.
