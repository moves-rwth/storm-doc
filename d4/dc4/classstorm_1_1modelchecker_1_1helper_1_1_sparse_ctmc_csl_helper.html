Computes the transient probabilities for lambda time steps.

Parameters
uniformizedMatrixThe uniformized transition matrix.
addVectorA vector that is added in each step as a possible compensation for removing absorbing states with a non-zero initial value. If this is not supposed to be used, it can be set to nullptr.
timeBoundThe time bound to use.
uniformizationRateThe used uniformization rate.
valuesA vector mapping each state to an initial probability.
epsilonThe precision used for computing the truncation points
Template Parameters
useMixedPoissonProbabilitiesIf set to true, instead of taking the poisson probabilities, mixed poisson probabilities are used.
Returns
The vector of transient probabilities.
