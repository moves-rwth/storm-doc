Detailed Description

template<typename ValueType>
class storm::dft::simulator::DFTTraceSimulator< ValueType >

Simulator for DFTs.

A step in the simulation corresponds to the failure of one BE (either on its own or triggered by a dependency) and the failure propagation through the DFT. The simulator also allows to randomly generate a next failure according to the failure rates.

template<typename ValueType >
SimulationTraceResult storm::dft::simulator::DFTTraceSimulator< ValueType >::simulateCompleteTrace ( double  timebound)

Perform a complete simulation of a failure trace by using the random number generator.

The simulation starts in the initial state and tries to reach a state where the top-level event of the DFT has failed. If this target state can be reached within the given timebound, the simulation was successful. If an invalid state (due to a restrictor) was reached, the simulated trace is invalid.

Parameters
timeboundTime bound in which the system failure should occur.
Returns
Result of simulation trace is (1) SUCCESSFUL if a system failure occurred for the generated trace within the time bound, (2) UNSUCCESSFUL, if no system failure occurred within the time bound, or (3) INVALID, if an invalid state (due to a restrictor) was reached during the trace generation.
