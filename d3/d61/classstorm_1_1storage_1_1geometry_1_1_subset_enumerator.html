Detailed Description

template<typename DataType = std::nullptr_t>
class storm::storage::geometry::SubsetEnumerator< DataType >

This class can be used to enumerate all k-sized subsets of {0,...,n-1}.

A subset is represented as a vector of ascending numbers. Example: (n=5, k=3) [0,1,2] --> [0,1,3] --> [0,1,4] --> [0,2,3] --> [0,2,4] --> [0,3,4] --> [1,2,3] --> [1,2,4] --> [1,3,4] --> [2,3,4] A filter can be given which should answer true iff it is ok to add a given item to a given subset. Example: (n=5, k=3, filter answers false iff 0 and 2 would be in the resulting subset): [0,1,3] --> [0,1,4] --> [0,3,4] --> [1,2,3] --> [1,2,4] --> [1,3,4] --> [2,3,4]
